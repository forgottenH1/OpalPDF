# OrbitPDF Ad Specifications

You can use this document to send to potential advertisers. It lists all the accepted formats and dimensions for our ad placements, following strict IAB Standards.

## 1. Sidebar Ads (Premium Placement)

These are fixed-size banners appearing on the left and right sides of the Tools and Guides pages.

| Device | Dimensions (Width x Height) | IAB Name | Placement |
| :--- | :--- | :--- | :--- |
| **Desktop** | **160px × 600px** | Wide Skyscraper | Left & Right Sidebar |
| **Mobile** | Not displayed | N/A | Hidden on mobile |

---

## 2. Header & Footer Banners (Responsive)

These are standard leaderboard banners.

### Desktop Version
| Dimensions (Width x Height) | IAB Name |
| :--- | :--- |
| **728px × 90px** | **Leaderboard** (Standard) |

### Mobile Version
This image will be automatically served to users on phones (<768px).

| Dimensions (Width x Height) | IAB Name |
| :--- | :--- |
| **320px × 50px** | **Mobile Leaderboard** (Standard) |

---

## 3. Supported File Formats

*   **Static Images:** `.jpg`, `.png`, `.webp`
*   **Animated Images:** `.gif`, `.webp` (animated)
*   **Transparent Backgrounds:** Supported
