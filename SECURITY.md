# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please do NOT report it publicly.

1.  Email your findings to `security@orbitpdf.com` (placeholder - please update).
2.  We will acknowledge your report within 48 hours.
3.  We will provide a timeline for the fix.

## Security Features
- **Client-Side Processing**: Files are processed in your browser and never uploaded to our servers (except for specific cloud-based tools if explicitly stated).
- **Content Security Policy**: Enforced Strict CSP.
- **HTTPS**: Forced via HSTS.
